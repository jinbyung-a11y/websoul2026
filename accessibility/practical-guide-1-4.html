1.4 명료성

콘텐츠는 사용자가 쉽게 보고 들을 수 있어야 합니다.

색에 무관한 콘텐츠 인식, 자동 재생 금지, 텍스트 명도 대비, 콘텐츠 간의 구분 등 시각·청각적 명료성을 확보합니다.

1.4.1. 색에 무관한 콘텐츠 인식

콘텐츠는 색에 관계없이 인식될 수 있어야 한다.

• 색상만으로 내용을 구분하거나 인식하도록 제공된 콘텐츠(그래프, 차트, 지도, 필수 입력 항목 등)를 제공한 경우

• 활성/비활성 상태를 색으로만 구분하는 경우

1.4.2. 자동 재생 금지

자동으로 소리가 재생되지 않아야 한다.

• 웹 페이지에서 자동적으로 재생되는 3초 이상의 배경 음 콘텐츠를 제공하는 경우

• 마우스 오버 또는 키보드 초점을 받아 자동적으로 배경 음이 3초 이상 실행되는 경우

1.4.3. 텍스트 콘텐츠의 명도 대비

텍스트 콘텐츠와 배경 간의 명도 대비는 4.5 대 1 이상이어야 한다.

• 보통 크기(18pt 미만, 또는 굵은 14pt 미만)의 일반 텍스트가 4.5:1을 만족하지 않은 경우

• 텍스트의 크기가 18pt 이상, 또는 굵은 14pt 이상의 텍스트가 3:1을 만족하지 않은 경우

1.4.4. 콘텐츠 간의 구분

이웃한 콘텐츠는 구별될 수 있어야 한다.

• 이웃한 콘텐츠를 구분하지 않은 경우
