2.4 쉬운 내비게이션

콘텐츠의 반복되는 영역은 건너뛸 수 있어야 합니다.

반복 영역 건너뛰기, 제목 제공, 적절한 링크 텍스트, 고정된 참조 위치 정보 등 사용자가 쉽게 원하는 콘텐츠를 찾을 수 있도록 합니다.

2.4.1. 반복 영역 건너뛰기

콘텐츠의 반복되는 영역은 건너뛸 수 있어야 한다.

• 건너뛰기 링크를 제공하지 않은 경우

• 건너뛰기 링크의 제공 방법이 적절하지 않은 경우

2.4.2. 제목 제공

페이지, 프레임, 콘텐츠 블록에는 적절한 제목을 제공해야 한다.

• 페이지에 제목을 제공하지 않은 경우

• 페이지에 제목을 적절하게 제공하지 않은 경우

• 프레임 요소에 제목을 제공하지 않은 경우

• 프레임 요소에 제목을 적절하게 제공하지 않은 경우

• 콘텐츠 블록에 제목을 제공하지 않은 경우

• 콘텐츠 블록에 제목을 적절하게 제공하지 않은 경우

2.4.3. 적절한 링크 텍스트

링크 텍스트는 용도나 목적을 이해할 수 있도록 제공해야 한다.

• 목적이나 용도를 알기 어려운 링크 텍스트를 제공한 경우

2.4.4. 고정된 참조 위치 정보

전자출판문서 형식의 웹 페이지는 각 페이지로 이동할 수 있는 기능이 있어야 하고, 서식이나 플랫폼에 상관 없이 참조 위치 정보를 일관되게 제공ㆍ유지해야 한다.

• 웹페이지로 제공되는 전자출판문서 콘텐츠에 대한 페이지 구분 정보를 제공하지 않은 경우

• 웹페이지로 제공되는 전자출판문서 콘텐츠에 대한 페이지 이동 기능을 제공하지 않은 경우

• 콘텐츠 변경(글꼴 변경, 간격 조절 등) 시 일관된 페이지 구분 정보를 제공·유지하지 않은 경우
